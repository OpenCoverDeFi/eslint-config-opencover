# ESLint Rules Documentation

This directory contains comprehensive documentation for all ESLint rules configured in this package.

## Rule Categories

- **[OpenCover Custom Rules](./opencover/README.md)** - Rules specific to OpenCover's coding standards
- **[TypeScript ESLint Rules](./typescript-eslint/README.md)** - TypeScript-specific linting rules with all recommended and overridden settings
- **[ESLint Rules](./eslint/README.md)** - Core JavaScript/TypeScript formatting and best practices
- **[Import Plugin Rules](./import/README.md)** - Import/export statement linting
- **[Stylistic Plugin Rules](./stylistic/README.md)** - Code formatting and style rules
- **[Unicorn Plugin Rules](./unicorn/README.md)** - Additional best practices
- **[Vitest Plugin Rules](./vitest/README.md)** - Test file specific rules (applied to `*.test.ts` files)
